# Дорожная Карта

## Суть
`carrd-v2` закрепил split theme/runtime contract: новые installs идут через `theme-runtime` + полный `theme-design-tokens-embed.html`, а `theme-core` / `theme-ui.css` оставлены только как controlled compatibility tails для существующих mutable `@main` consumers. Тег `v2.1.0` опубликован и вживую подтверждён на `main-template`/`mini.crd.co` 2026-07-07 — весь сайт на version-pinned `@2.1.0`, ноль `@main`-ссылок, включая add-on plugins. Следующий шаг — продолжать миграцию на version-pinned runtime path для `faktura` -> export/install в `faktura-app`.
